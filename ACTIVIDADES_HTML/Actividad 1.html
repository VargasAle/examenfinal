HTML

HTML son las siglas de Hypertext Markup Language que puede traducirse commo lenguaje de marcas o marcado de hípertexto.

El lenguaje HTML se emplea para crear paginas web. Es muy facol ver el códigp HTML de una pagina web, la opción exacta cambia de un anvegador a otro y también puede cambiar de una versión a otra de un mismo navegador, pero suelen tener un nombre similar

Historia de HTML

Los inicios del lenguaje HTML se remontan al año 1990, cuando Tim Berners-Lee creó la primera pagina web.
